RQ1

Modeling User Behavior

Modeling User Behavior

Searcher model framework proposed by Baskaya et al. (2013).

Modeling search as a sequence

User search is a sequence of actions.
This can be modeled using a Markov Chain.

State: \(s_t\) Action: \(a_t\) Transition: \(P(s_{t+1} | s_t)\)

Abstraction of transition probabilities between user search actions. Baskaya et al. (2013).

Problem. Real user behavior depends on more than just the last action.

Basic models lack context

Intuition. Bundle historical context into the state.

\(s_t = (\textcolor{#3498db}{q_t}, \textcolor{#2ecc71}{a_{t-1}}, \textcolor{#f39c12}{q_0}, \dots)\)

\(q_t\): current query

\(a_{t-1}\): previous action

\(q_0\): session's initial query

We can now assign probabilities conditioned on context from \(s_t\)1.

\( P(a_t \mid \textcolor{#3498db}{q_t}, \textcolor{#2ecc71}{a_{t-1}}, \textcolor{#f39c12}{q_0}) = \frac{N(\textcolor{#3498db}{q_t}, \textcolor{#2ecc71}{a_{t-1}}, \textcolor{#f39c12}{q_0}, a_t)}{N(\textcolor{#3498db}{q_t}, \textcolor{#2ecc71}{a_{t-1}}, \textcolor{#f39c12}{q_0})} \)

Where \(N(...)\) is the frequency of observation in user logs.

1Saber Zerhoudi et al. "Simulating User Search Behaviour in Digital Libraries". IRCDL 2022.

RQ2

Evaluating Realism

Evaluating Realism

Problem. Traditional metrics track effectiveness‚Äînot behavioral realism.
A simulator may score high... while acting in a completely unrealistic way.

Shift in focus. Not "Was the session productive?"‚Äîbut "Was the search pattern realistic?"

Can a classifier tell the difference?

Approach 1 The Turing Test

Idea: Train a classifier to distinguish real user sessions from simulated ones.

Hypothesis. A realistic simulation should confuse the classifier, leading to low accuracy.

Verification. We apply the KS-2 test: checks whether distributions of key features (session length, click positions) are statistically indistinguishable.

Distance between behaviors

Approach 2 Fr√©chet Distance

Fr√©chet Distance8 quantifies how similar two curves are.

Fr√©chet Distance measures the similarity between two curves.

Evaluation of Text-to-Image (TTI) generative models with Fr√©chet distance.

Evaluation of IR systems with Fr√©chet distance.

We treat real vs. simulated sessions as trajectories in a high-dimensional space.

Goal. A lower Fr√©chet Distance → simulated behavior is closer to the real behavior.

8Saber Zerhoudi, Michael Granitzer "Beyond Conventional Metrics: Assessing User Simulators in Information Retrieval". IIR 2024.

Ranking simulators

New metrics. They rank simulators by realism‚Äîconsistently and reliably.

Lower bars (F-Measure) and lower red squares (KS-2 D-statistic) → better realism.

Takeaway. Advanced models‚Äîcontextual, query-change, CACSM‚Äîare demonstrably more realistic.

Summary

  • Traditional metrics are insufficient for evaluating simulator realism. We proposed two new approaches:
    • A Turing Test-style classification method to assess authenticity.
    • Fr√©chet Distance to quantify the distributional distance between real and simulated behaviors.

Conclusion

In this thesis, we advanced user simulation for IR evaluation‚Äîacross four dimensions:

  • Complex Behavior Models. More realistic mathematical (Contextual Markov) and cognitive (CACSM) models.
  • Generative AI Simulators. An autonomous agent (Agent4DL) that can reason, learn, and generate behavioral data.
  • Novel Validation Methods. New measures of behavioral realism (Classification-KS2, Fr√©chet Distance).
  • A Unified Simulation Framework. A modernized, open-source tool for the community (SimIIR 3).

Future Outlook

The models and frameworks developed in this thesis provide a foundation for building the next generation of intelligent, user-centric search experiences.
