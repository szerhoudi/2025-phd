Conclusion

In this thesis, we focused on:

  • Modeling complex search behavior using MDP and CACSM.
  • Enhancing simulation with Generative AI using Agent4DL architecture.
  • Evaluating simulation realism using Classification-KS2 and Fréchet Distance.
  • Providing a unified simulation framework for search experiments (SimIIR3.0).
